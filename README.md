# hashz


### Install & Run on PC

```
git clone https://github.com/hashz0/hashz
cd hashz
pip3 install kivy
python3 hashz.py
```
### Install & Run on Android
Download precompiled binary from releases tab or build it with buildozer

[I want CLI version!](https://github.com/hashz0/hashz-cli)
